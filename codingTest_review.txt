- 뒤집어진 숫자
    
    입력된 숫자를 문자열로 바꿔 문자열의 총 길이를 구한 후, 그 길이를 가지고 배열을 선언한다.
    
    입력된 숫자를 while문을 사용해서 하나씩 나눈다.
    
    while문 안에서 10으로 나눈 나머지를 배열에 저장하고 들어온 숫자를 10으로 나눈다.
    
    1의 자리부터 앞의 자리 수까지 거꾸로 숫자를 구할 수 있다.
    
    배열에 거꾸로 구한 숫자를 문자열로 사용해서 하나로 합친 후 문자열로 바꾼 매개변수 숫자와  equals문으로 비교해서 결과 값을 리턴한다.
    
    - 문자열로 합치지 않고 int형으로 사용하려면
        
         int num = 0;
        
         for (int i =0 ; i<array.length; i++){
        
        num = num*10;
        
        num += array[i];
        
        }
        
   
   
- 잃어버린 카드
    
    해쉬맵을 사용한다.
    
    put()을 사용하여 키에는 카드 번호를 넣고, 키의 값에는 getOrDefault(카드번호, 0)+1을 사용해
    
    기본적으로 1씩 값을 넣고 중복되는 값이 있으면 1씩 증가시켜 값을 수정한다.
    
    그래서 2개의 카드가 온전하게 있으면 키 값은 2가되고 잃어버린 카드의 키 값은 1이 된다.
    
    키 값이 1인 것을 찾아 배열에 넣고, 그 배열을 정렬해서 리턴한다.
    
    
    
    
- 로봇청소기
    
    robot += area[r][c];
    
    area[r][c] = 0; // 처음 자리 청소하기
    
    명령어가 go일때 ⇒ 방향에 따라 이동을 한다. 
    
    방향이 1이면 북쪽으로 이동 r-1, c
    
    방향이 2이면 동쪽으로 이동 r, c+1
    
    방향이 3이면 남쪽으로 이동 r+1, c
    
    방향이 4이면 서쪽으로 이동 r, c-1
    
    만약 이동할 곳의 r,c가 0보다 작거나 면적의 수보다 크거나 area[r][c]가 0보다 작으면
    
    이동하지 못하고 원 상태로 돌아간다.
    
    모두 만족하면 
    
    robot += area[r][c]; //청소기 먼지 추가
    
    area[r][c] = 0; //장소에 먼지가 사라진다.
    
    명령어가 right, left일떄 ⇒ 방향을 바꾼다. 방향이 1보다 작으면 4로 이동, 4보다 크면 1로 이동해 방향을 순환하게 한다.
